# smart-rental-price-prediction-analysis
Developed a Smart Rental Price Prediction system using Python and Machine Learning on 1500+ housing records. Performed data cleaning, EDA, and feature engineering to identify key factors influencing rent. Built predictive models and generated insights using data visualization tool.

## 📌 Overview
This project predicts house rental prices using Machine Learning on 1500+ records.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model

## 📊 Key Insights
- Mumbai has highest rent (~₹53K)
- Rent increases with BHK and Size
- Furnished houses cost more
- Strong correlation between Size & Rent

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
